Visual validation service

Why? It is hard to catch visual errors just by looking at two sites, or browsers at the same time.

Api Usage:

curl --request POST \
 --url /api/crosssite \
 --header 'content-type: application/json' \
 --data '{
  "browser":"chrome",
  "urls":{
   "base":"https://www.google.com",
   "targets":[
    "https://www.google.com","https://www.google.com?q=hola"]
  }
}'
